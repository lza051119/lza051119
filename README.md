# Hi, I'm Zi'an Liang

I am an undergraduate student in the **School of Cyberspace Security** at **Beijing University of Posts and Telecommunications (BUPT)**.

**B.E. candidate, 2023–2027**

I am interested in building and evaluating secure, trustworthy AI agent systems. My current work spans agent-harness security evaluation, LLM safety, enterprise AI agent platforms, and applied cybersecurity.

## Research Interests

- AI agent and large language model security
- Security evaluation of coding agents and agent harnesses
- Prompt injection, persistent poisoning, and tool/MCP security
- Memory and workspace security for agentic systems
- Web agent privacy and multimodal agent security
- Network security, attack tracing, and trusted computing

## Research & Engineering Experience

### AgentOS / AgentHub — Enterprise AI Agent Platform

- Contributed to the design and development of an enterprise-oriented platform for deploying, integrating, and governing AI agents.
- Worked with plugin architecture, Gateway/HAL components, Keycloak-based identity management, OIDC, RBAC/ACL authorization, credential management, and enterprise system integration.
- Participated in requirements analysis, solution design, deployment testing, and technical validation for enterprise proof-of-concept environments.

### Safety Bench — Agent-Harness Security Evaluation

- Participated in security evaluation research for coding agents and agent harnesses, focusing on prompt injection, persistent poisoning, tool/MCP abuse, and memory/workspace poisoning.
- Contributed to benchmark and evaluation framework design, including task abstraction, oracle-based assessment, risk scoring, experiment runners, trace analysis, and multi-harness adaptation.
- Studied how untrusted inputs can propagate across memory, skills, tools, workspaces, and sessions in agentic systems.

### LLM Safety and Web Agent Security

- Investigated automated jailbreak methods including GCG, AutoDAN, PAIR, and AdvPrompter, with attention to attack mechanisms and evaluation methodology.
- Explored environmental injection attacks and personally identifiable information leakage risks in multimodal web agents.
- Worked with safety evaluation datasets and metrics to examine the balance between model security and utility.

### TraceX — Attack Behavior Tracing and Analysis

- Developed a layered analysis pipeline covering Sigma-based detection, ATLAS-oriented semantic abstraction, graph construction, and path tracing.
- Modeled heterogeneous security logs as entity-relation graphs and used graph traversal to reconstruct potential attack chains.
